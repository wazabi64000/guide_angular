Angular - Installation & Configuration

10 Cours Théoriques Approfondis

1. Installer Node.js et npm : Angular nécessite Node.js >= 14 et npm pour gérer les packages.

2. Installer Angular CLI : npm install -g @angular/cli. CLI fournit les commandes pour générer projets, composants, services, modules.

3. Créer un projet Angular : ng new mon-projet. Explication des options : routing (oui/non), style (CSS/SCSS), package manager.

4. Démarrer le serveur de développement : ng serve. Serveur live reload pour visualiser les changements instantanément.

5. Structure du projet : comprendre src/app (composants), assets (images, fichiers statiques), index.html, angular.json.

6. Modules Angular : NgModule organise composants, services, pipes, directives. AppModule est le module racine.

7. Environnements : environments/environment.ts pour dev/prod, utile pour API endpoints et configurations.

8. TypeScript : Angular utilise TypeScript strict pour typage, interfaces et classes.

9. Installer packages supplémentaires : RxJS pour gestion Observables, Angular Material pour UI, ou d’autres libs selon besoin.

10. Compilation production : ng build --prod minifie et optimise l’application pour déploiement.

Exemples Commentés


/* 1. Créer projet Angular */
ng new mon-projet
/* Options : routing: yes/no, CSS ou SCSS */

/* 2. Démarrer serveur dev */
cd mon-projet
ng serve
/* Live reload sur http://localhost:4200 */

/* 3. AppModule de base */
import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule],
  bootstrap: [AppComponent]
})
export class AppModule {}

/* 4. Ajouter un package via npm */
npm install @angular/material
