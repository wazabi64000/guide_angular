Mini-Projet Angular : Guide Complet Pas-à-Pas

Étape 1 : Installer Angular et créer le projet

Objectif : Avoir un projet Angular fonctionnel prêt à coder.

  • Installer Node.js et npm
  • Installer Angular CLI : npm install -g @angular/cli
  • Créer projet Angular : ng new mini-projet, activer routing et SCSS
  • Lancer le serveur : ng serve
ng new mini-projet --routing --style=scss
cd mini-projet
ng serve

Étape 2 : Créer des composants

Objectif : Organiser ton application en modules et composants réutilisables.

  • Composant formulaire : ng generate component UserForm
  • Composant liste utilisateurs : ng generate component UserList
  • Composant carte utilisateur : ng generate component UserCard
ng generate component UserForm
ng generate component UserList
ng generate component UserCard

Étape 3 : Configurer le routage

Objectif : Naviguer entre les pages via URL.

  • Définir routes dans app-routing.module.ts :
    • const routes: Routes = [
  { path: 'form', component: UserFormComponent },
  { path: 'list', component: UserListComponent },
  { path: '', redirectTo: '/form', pathMatch: 'full' }
];
  • Ajouter <router-outlet></router-outlet> dans app.component.html
imports: [RouterModule.forRoot(routes)]

Étape 4 : Créer le formulaire utilisateur

Objectif : Récupérer dynamiquement les informations saisies.

  • Importer FormsModule dans AppModule
  • Lier les champs avec [(ngModel)]
    • <form (ngSubmit)="onSubmit()">
  <input [(ngModel)]="name" name="name" placeholder="Nom" required />
  <input [(ngModel)]="email" name="email" placeholder="Email" required />
  <button type="submit">Ajouter</button>
</form>
name = '';
email = '';
onSubmit() {
  console.log('Nom:', this.name, 'Email:', this.email);
}

Étape 5 : Créer le service API

Objectif : Centraliser l’accès aux données utilisateurs.

  • Créer UserService et utiliser fetch() ou HttpClient
    • @Injectable({providedIn:'root'})
export class UserService {
  apiUrl = 'http://localhost:5000/users';
  getUsers(){ return fetch(this.apiUrl).then(res => res.json()); }
  addUser(user){ return fetch(this.apiUrl,{
    method:'POST',
    headers:{'Content-Type':'application/json'},
    body:JSON.stringify(user)
  }).then(res => res.json()); }
}
// UserFormComponent
this.userService.addUser({name:this.name,email:this.email})
  .then(u => console.log('Utilisateur ajouté', u));

Étape 6 : Afficher la liste des utilisateurs

Objectif : Récupérer et afficher les utilisateurs dynamiquement.

  • Récupérer données dans ngOnInit :
    • users = [];
loading = false;
error = '';

ngOnInit() {
  this.loading = true;
  this.userService.getUsers()
    .then(data => this.users = data)
    .catch(err => this.error = 'Impossible de charger les utilisateurs')
    .finally(() => this.loading = false);
}
  • Afficher avec *ngFor :
    • <div *ngFor="let user of users">
  <h3>{{user.name | titlecase}}</h3>
  <p>{{user.email | lowercase}}</p>
</div>
// Vérifier que la liste se met à jour après ajout d'utilisateur
