Angular - Composants & Binding

10 Cours Théoriques Approfondis

1. Créer un composant : ng generate component NomComposant. Chaque composant est autonome avec template, style et logique.

2. Templates HTML et CSS : séparation claire pour lisibilité et réutilisation.

3. Interpolation : {{ variable }} pour afficher dynamiquement des données.

4. Property Binding : [attr]="valeur" pour lier des propriétés HTML aux variables TypeScript.

5. Event Binding : (click)="fonction()" pour gérer événements utilisateur.

6. Two-way Binding : [(ngModel)]="variable" pour synchroniser input et variable automatiquement.

7. Input & Output : @Input() pour passer des données à un composant enfant, @Output() + EventEmitter pour envoyer vers le parent.

8. Communication parent-enfant et enfant-parent pour architecture modulaire.

9. Réutilisation : composants configurables via @Input() pour éviter duplication de code.

10. Bonnes pratiques : composants petits et concentrés sur une seule responsabilité.

Exemples Commentés


/* 1. Créer composant UserCard */
ng generate component UserCard

/* 2. Interpolation */
<h3>{{ user.name }}</h3>

/* 3. Property Binding */
<img [src]="user.avatar" alt="{{user.name}}" />

/* 4. Event Binding */
<button (click)="deleteUser(user.id)">Supprimer</button>

/* 5. Two-way Binding */
<input [(ngModel)]="user.name" placeholder="Modifier nom" />

/* 6. Input pour recevoir data */
@Input() user;

/* 7. Output pour notifier parent */
@Output() deleted = new EventEmitter();
delete() {
  this.deleted.emit(this.user.id);
}

/* 8. Utilisation composant enfant dans parent */
<app-user-card *ngFor="let u of users" [user]="u" (deleted)="removeUser($event)"></app-user-card>
